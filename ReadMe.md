# このソフトウェアについて #

HelloWindowForMinGWは私個人が学習目的で作成したソフトウェアである。

MinGWのgccでHelloWorldした。（WindowApplication版）

# 開発環境 #

* Windows XP Pro SP3 32bit
* MinGW
    * msys 1.0
    * gcc 4.6.2
